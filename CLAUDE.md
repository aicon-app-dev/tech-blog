# テックブログ

株式会社Aiconのテックブログ。主に採用向けの技術発信。

## ディレクトリ構成

- `drafts/` — 下書き・レビュー・改善案
- `published/` — 公開済み（ファイル名: `YYYY-MM-DD_タイトル.md`）
- `assets/` — 記事用の図表画像
- `screenshots/` — GitHub等のスクリーンショット
- `seo-reports/` — SEO分析レポート
- `take_screenshots.py` — スクリーンショット取得スクリプト

## 記事の方針

- ターゲット: エンジニア採用候補者
- トーン: 実態ベース。盛らない、でも刺さるファクトは出す
- 内容例: AI自律開発の実態、ハーネスエンジニアリング、開発速度の定量データ
- 注意: 「一発生成できました」系のインプレ稼ぎにならないように。ちゃんとした実態を伝える
